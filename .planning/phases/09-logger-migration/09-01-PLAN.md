<!-- Etabliertes Logger-Pattern aus bestehenden ViewModels -->
```swift
import os
private let logger = Logger(subsystem: Bundle.main.bundleIdentifier ?? "ValetudiOS", category: "TypeName")
```

Privacy-Konventionen:
- Fehlermeldungen: `logger.error("Beschreibung: \(error.localizedDescription, privacy: .public)")`
- Sensible Daten (IPs, Hostnamen): `privacy: .private`
- Debug-Info: `logger.debug("...")`
